# Számításelmélet alapjai 1 — Vizsgafelkészítő

Interaktív, egyfájlos tanulóoldal a *Formális nyelvek és automaták* (A számításelmélet alapjai 1) vizsgához. A teljes vizsgakérdés-bank megtanítva, mind a három feladatrészre, magyarázatokkal.

**Élő oldal:** https://bummafejbe.github.io/Szamelm/ *(a GitHub Pages bekapcsolása után — lásd lentebb)*

## Mit tartalmaz

- **1. rész – Igaz/Hamis** (93 állítás, 8 témakör) — mindegyiknél a *miért*.
- **2. rész – Rövid válaszok** (30 feladat) — definíciók, konstrukciók, „Miért?” magyarázattal.
- **3. rész – Kifejtős** (16 sablon) — normálformák, eljárások, CYK, Bar-Hillel lemmák.
- **Fogalomtár** — ~45 egymásra épülő fogalom, „Ehhez kell:” kereszthivatkozásokkal, plusz összefoglaló táblázatok.

## Funkciók

- **Lenyitható koncepció-ablakok** a kérdéseknél: minden témakörnél ott vannak a megoldáshoz szükséges fogalmak csipekként — kattintásra kinyílnak.
- **Gyakorló mód**: elrejti az Igaz/Hamis válaszokat önteszthez.
- **Keresés** a kártyák között.
- **Sötét, letisztult, reszponzív** dizájn — telefonon is jól néz ki.
- Teljesen **offline** is működik (egyetlen HTML-fájl).

## Helyi futtatás

Töltsd le az `index.html`-t, és nyisd meg a böngésződben (dupla kattintás).

## GitHub Pages bekapcsolása

A repo **Settings → Pages** menüjében a *Source* legyen **`Deploy from a branch`**, az ág **`main`**, a mappa **`/ (root)`**. Pár perc múlva elérhető a fenti címen.
